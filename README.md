# Portfolio Next.js + Tailwind (Milton)

Plantilla lista para desplegar en **Vercel**.

## Pasos

1. Instalar dependencias
```bash
npm install
```

2. Ejecutar en desarrollo
```bash
npm run dev
```

3. Editar tus datos en `data/profile.js` (nombre, email, links, proyectos).

4. Subir a GitHub (opcional pero recomendado)
```bash
git init
git add .
git commit -m "init portfolio"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/mi-portfolio.git
git push -u origin main
```

5. Desplegar en **Vercel**
- Ingresá a https://vercel.com/ con tu cuenta de GitHub.
- **New Project** → elegí tu repo → **Deploy**.
- Listo: tu sitio quedará online.

## Tecnologías
- Next.js (App Router)
- TailwindCSS
- Framer Motion
